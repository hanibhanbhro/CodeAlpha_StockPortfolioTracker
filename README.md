# 📊 Stock Portfolio Tracker (Python)

## 📌 Description
This is a simple Python project that calculates total stock investment based on user input and predefined stock prices.

## ✨ Features
- Add stocks and quantity
- Calculates total investment
- Simple and beginner-friendly CLI program

## 🛠 How to Run
1. Open the file in Jupyter Notebook or Google Colab
2. Run all cells
3. Enter stock names and quantities
4. Type 'done' to finish

## 📚 Concepts Used
- Dictionaries
- Loops
- User Input
- Basic Python logic
